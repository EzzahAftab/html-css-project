# Amazon Clone (HTML & CSS)

This project is a **front-end clone of the Amazon website**, built using only **HTML and CSS**.  
It replicates the design and layout of Amazon’s homepage, including navigation bars, product sections, and footer elements.

---

## Features

-  **Navigation Bar** — includes logo, delivery location, search bar, and cart icon.  
-  **Hero Section** — highlights promotional messages and links.  
-  **Shop Section** — displays multiple product categories such as electronics, fashion, and personal care.  
-  **Footer Section** — designed similar to Amazon’s real site with multiple link panels.  
-  **Fully styled with CSS** — responsive and visually appealing design.  
-  **Font Awesome Icons** integrated for a modern UI.

---

##  Technologies Used

- **HTML5**
- **CSS3**
- **Font Awesome (CDN)** for icons
- Local image assets (`.jpg` files)
